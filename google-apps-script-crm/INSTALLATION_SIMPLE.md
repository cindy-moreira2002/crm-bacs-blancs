# Installation simple en un seul copier-coller

Google Sheet :

https://docs.google.com/spreadsheets/d/1AcjIQCSIlaHK-7-wPKzhOJzrhzDXjdNmBlthHgVg0Jo/edit

## Étapes

1. Ouvrez le Google Sheet.
2. Allez dans `Extensions > Apps Script`.
3. Supprimez le contenu déjà présent dans `Code.gs`.
4. Ouvrez le fichier local `Code.gs` dans ce dossier.
5. Copiez tout son contenu.
6. Collez-le dans `Code.gs` dans Apps Script.
7. Cliquez sur Enregistrer.
8. Dans la liste des fonctions, choisissez `initializeCrm`.
9. Cliquez sur Exécuter.
10. Acceptez les autorisations Google.
11. Revenez dans le Google Sheet et rechargez la page.

Le menu `Prospection MDB` apparaîtra dans la barre de menu.

Important : le CRM reste en mode test par défaut. Remplissez `CONFIGURATION`, surtout `Adresse e-mail de test`, avant de créer des brouillons Gmail.
